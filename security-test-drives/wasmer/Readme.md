# Wasmer GO

## Install GO

download GO: https://golang.org/doc/install

```bash
cd /usr/local
sudo cp ~/go**tar.gz .
sudo tar -xvzf go**tar.gz
export PATH=$PATH:/usr/local/go/bin
```

## Installation

```bash
go get github.com/wasmerio/wasmer-go/wasmer
go mod init
go build .
```
